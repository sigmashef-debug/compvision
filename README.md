# CounterPose – счётчик отжиманий, приседаний и других упражнений по позе человека

CounterPose — это приложение, которое в реальном времени анализирует позу человека с веб-камеры и автоматически считает количество повторений упражнений (отжимания, приседания, планка, подъёмы рук и т.д.).

---

## Возможности

-  **Работа с веб-камерой**
-  **Оценка позы** (keypoints / skeleton) с помощью моделей компьютерного зрения  
-  **Автоматический подсчёт повторений**:
  - Отжимания  
  - Приседания  
  - (при желании — любые другие упражнения, где можно определить фазу движения)  
-  **Онлайн-индикаторы**:
  - Текущий счёт повторений  
  - Фаза упражнения (вверх/вниз)  
-  **Сохранение логов**: время выполнения, количество повторений, тип упражнения
